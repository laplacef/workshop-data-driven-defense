## The Setting

You are an analyst at an industrial company with many edge devices and complicated network infrastructure. Recently, the company experienced suspicious traffic patterns that disrupted operations and raised concerns about potential cyber threats. Our task? Implement a machine learning-based solution to analyze, identify, and classify malicious activities and potential threats that can lead to data breaches.

They've provided you with a dataset containing network traffic logs, which you'll use to build a classification model for intrusion detection. You'll preprocess the data, select a suitable machine learning algorithm, and evaluate its performance. By the end of this workshop, you'll have a working prototype for an intrusion detection system that can help classify network traffic for the company's IIoT devices and protect against cyber threats.

## The Approach

1. **[Exploratory Data Analysis](data_analysis.ipynb)**: Explore the dataset to understand its structure and characteristics.
2. **[Preprocessing, Training, and Evaluation](model_analysis.ipynb)**: Clean and preprocess the dataset, select a suitable machine learning algorithm, and evaluate its performance.
